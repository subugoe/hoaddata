# Hybrid Journals: Venues Metadata

This datasets contains information about journals. These journals come
from the Journal Checker Tool.

## Usage

``` r
jct_oalex_venues
```

## Format

An object of class `tbl_df` (inherits from `tbl`, `data.frame`) with
12966 rows and 3 columns.
